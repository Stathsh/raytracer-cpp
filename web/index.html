Ray Tracer

An interactive ray tracer with a GUI built in C++17. Adjust spheres, camera, and materials in real time, then render scenes with reflections, shadows, and anti-aliasing.

Features

Interactive GUI

Adjust sphere positions, colors, reflectivity, and camera in real time with a full ImGui interface.

Live Preview

Watch the render progressively build line-by-line with a live texture preview in the GUI window.

Reflections

Recursive ray bouncing for mirror-like surfaces with per-material reflectivity control.

Shadows & Lighting

Multiple light sources with hard shadows, Blinn-Phong shading, and Reinhard tone mapping.

Anti-aliasing

Configurable supersampled AA from 1x to 64x samples per pixel for buttery-smooth edges.

BMP Export

Save your render to a BMP file at any resolution up to 4K. CLI version also available for scripting.

Getting Started

Requirements: The GUI app needs libsdl2 and OpenGL drivers installed. On Ubuntu/Debian: sudo apt install libsdl2-2.0-0. Most desktop Linux distros include these.
# GUI version — interactive window
chmod +x raytracer-gui
./raytracer-gui

# CLI version — render from command line (no deps needed)
chmod +x raytracer-cli
./raytracer-cli -w 1920 -h 1080 -s 4 -o render.bmp
# Build from source (needs SDL2 dev + ImGui)
sudo apt install libsdl2-dev libgl-dev
g++ -O3 -std=c++17 -I. main_gui.cpp imgui*.cpp imgui_impl_*.cpp $(sdl2-config --cflags --libs) -lGL -lm -lpthread
