Ray Tracer

An interactive desktop app for ray tracing 3D scenes. Adjust spheres, camera, and materials in a full GUI, then render with reflections, shadows, and anti-aliasing. Built with C++ and Electron.

Features

Full GUI

Interactive controls for camera, spheres, materials, and render settings. No command line needed.

Live Preview

Watch the render build progressively line-by-line with real-time preview in the app window.

Reflections & Shadows

Recursive ray bouncing for mirror surfaces and hard shadow casting from multiple light sources.

Per-Sphere Editing

Change position, radius, color, reflectivity, and shininess for each sphere individually.

Anti-aliasing

Configurable supersampled AA from 1x to 64x samples per pixel for smooth, clean edges.

Export to BMP

Save renders at any resolution up to 4K using the native file dialog on your system.

How to Run

# 1. Download the zip for your Mac
# Apple Silicon (M1/M2/M3): RayTracer-mac-apple-silicon.zip
# Intel Mac: RayTracer-mac-intel.zip

# 2. Extract the zip (double-click in Finder)

# 3. Open "Ray Tracer.app"
# If blocked: right-click > Open > click Open

# 4. Click "Render" and enjoy!
# 1. Download RayTracer-linux.AppImage

# 2. Make it executable and run
chmod +x RayTracer-linux.AppImage
./RayTracer-linux.AppImage

# That's it! No installation required.
